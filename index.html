¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje HTML (HyperText Markup Language), que se usa para crear páginas web.

Estructura básica de HTML5

<!DOCTYPE html> <html> <head> <title>Mi página</title> </head> <body> </body> </html>

Etiquetas principales

h1 - h6

Los títulos en HTML se utilizan para definir encabezados o secciones dentro de una página web.

p

Los párrafos en HTML se utilizan para organizar y mostrar texto en bloques dentro de una página web.

img

Las imágenes en HTML se utilizan para mostrar contenido visual dentro de una página web, como fotos, ilustraciones o gráficos.

a

Los enlaces en HTML (también llamados hipervínculos) permiten conectar una página web con otra, o dirigirse a diferentes partes de un mismo sitio.

button

botones en HTML son elementos que permiten al usuario realizar acciones, como enviar un formulario, limpiar datos o ejecutar alguna función.

ul / ol / li

las listas se utilizan para organizar información de forma ordenada o desordenada dentro de una página web.

Formularios en HTML5

Los formularios en HTML son una forma de recopilar información del usuario dentro de una página web. Se utilizan, por ejemplo, para registrarse, iniciar sesión, enviar comentarios o realizar compras. Los datos que el usuario introduce se pueden enviar a un servidor para procesarlos.

Multimedia en HTML5

Hay nuevas etiquetas semánticas que facilitan la estructura de los documentos HTML y que reemplazarán a etiquetas de versiones anteriores

Imágenes

El elemento de imagen HTML img representa una imagen en el documento..

Audio

El elemento audio se usa para insertar contenido de audio en un documento HTML o XHTML

Video

En HTML5 se introduce la interesante posibilidad de mostrar videos directamente desde nuestro navegador

Accesibilidad

Accesibilidad para personas con discapacidad: El atributo ALT permite que los lectores de pantalla.

Conclusión

HTML5 simplifica la creación de sitios web más interactivos y accesibles a la vez que mejora la experiencia del usuario y facilita el trabajo de los desarrolladores.
